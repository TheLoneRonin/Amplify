### Development

#### `yarn build`

Builds the smart contract and outputs it to `dist`.

#### `yarn test`

Runs unit tests in `test`.

#### `yarn deploy`

Deploys the smart contract assuming your JWK wallet is under the path `./arweave.creds.json`.
